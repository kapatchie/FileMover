# FileMover
The goal of this application is to make sorting your downloads folder easier by allowing you to create a folder on your desktop and move the files there based on the type 

The application supports these Extensions at this point:

 [1] Audio: aif,cda,mid,midi,mp3,mpa,ogg,wav,wma,wpl
 [2] Video: 3g2,3gp,avi,flv,h264,m4v,mkv,mp4,mov,mpg,mpeg,rm,swf,vob,wmv
 [3] Compressed Data: 7z,arj,deb,pkg,rar,rpm,tar.gz,z,zip
 [4] Executables: apk,bat,bin,cgi,pl,com,exe,gadget,jar,msi,py,wsf
 [5] Image : ai,bmp,gif,ico,jpeg,jpg,png,ps,psd,svg,tif,tiff
 
 It is not currently possible to change the start folder it will always take from download to your desktop where you can specify a folder using the following format:
 Example\Example no need to specify that the desktop is the target

There is a file loading and saving sytem in the code it is Not yet implemented, this will remember where you stored a file the last time and move the same type of file there 
once implimented
